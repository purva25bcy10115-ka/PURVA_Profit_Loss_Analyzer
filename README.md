# PURVA_Profit_Loss_Analyzer

##Introduction

Company Profit &amp; Loss Analyzer is a Python-based project that calculates monthly revenue, expenses, and profit/loss using user input. It generates graphical reports using bar charts and stores company financial data in CSV format for analysis and record management.

# Company Profit & Loss Analyzer

## Author: PURVA KATARIA


## Project Overview
Company Profit & Loss Analyzer is a Python-based business analysis project that calculates monthly profit or loss using production and financial data entered by the user. The project also generates a graphical bar chart representation of company performance.

## Features
- Monthly profit and loss calculation
- Revenue and expense analysis
- Bar graph visualization
- CSV file data storage
- Simple and clean Python code
- User-friendly console interface

## Technologies Used

## How to Run
1. Open project in VS Code
2. Activate virtual environment
3. Install matplotlib using:
   pip install matplotlib
4. Run the program:
   python main.py
